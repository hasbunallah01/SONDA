## Adding a new component

```bash
pnpm dlx shadcn@latest add <component-name>
```

## Conventions

- Each primitive is a single file with no business logic.
- Variants are expressed with `class-variance-authority` (CVA).
- Tailwind classes are merged with `cn()` from `@/lib/utils`.
